## Easy to Use

```typescript
// Define your schema
const User = Table.normal({
  name: 'user',
  fields: {
    name: Field.string(),
    email: Field.string({ 
      assert: 'string::is::email($value)' 
    }),
    createdAt: Field.datetime({ 
      default: 'time::now()' 
    })
  }
});

// Type-safe queries
const users = await User.select(db, {
  where: 'email LIKE $pattern',
  limit: 10
}, { pattern: '%@example.com' });
```
